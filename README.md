# PGCET 2026 React Vite Project

A complete React + Vite admission portal inspired by the uploaded video UI.

## Features
- Responsive Home page with hero banner, quick actions, information center, dates and helpdesk
- Working navigation: Notification, Syllabus, Exam Pattern, User Guide, Regional Centers, Important Dates, Contact
- Working quick links: Registration, Application Form, Track Application, Payment Status, Hall Ticket Download
- Downloadable sample PDF links from `public/downloads`
- Local form submission demo using browser localStorage
- Mobile menu and clean corporate blue/orange styling

## Run locally
```bash
npm install
npm run dev
```

Then open the local URL shown by Vite.

## Build
```bash
npm run build
npm run preview
```
